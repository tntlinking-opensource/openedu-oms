## 🏠【关于我们】

![天天开源](https://open.tntlinking.com/assets/logo-b-BzFUYaRU.png) 

天天开源致力于构建开放共赢的平台，推动安全高效的开源应用普及。我们始终秉持“开源、众包、共享”的理念，致力于为医疗、教育、中小企业等行业提供优质的开源解决方案。

天天开源聚焦医疗、企业、教育三大行业信息化市场，现已发布OpenHIS、OpenCOM、OpenEDU系列开源软件产品。我们通过建立开源生态，联合生态伙伴共同打造创新行业协作模式，让数字化普惠、可信、安全。

天天开源的前身是新致开源，最早于2022年6月发布OpenHIS开源医疗，于2023年6月发布OpenCOM开源企业。2025年7月，新致开源品牌升级为天天开源。我们将持续践行开源精神，期待成为全球开源生态的引领者。

# 项目介绍

OpenEDU校园迎新系统是一款适用于中等教育院校（中职、中专、职高）、高等教育院校（高职、大专、本科以上院校）、各类技师学院的校园迎新系统。系统涵盖通知书打印、现场迎新、信息采集、移动迎新等核心功能。系统移动端集成微信服务，学生可通过手机办理登记，实现“线下+线上”一体化迎新，助力招生部门降低工作负荷，提升管理精细化水平与学生满意度。

系统涵盖通知书打印、现场迎新、信息采集、移动迎新等核心功能，显著提升迎新效率与学生体验。系统支持迎新现场实时统计，自动刷新报到进度与环节办理人数；迎新首页集成登录、新闻、流程指引等模块，学生可一站式完成信息填报、绿色通道申请、军训服装选择等操作。后台具备灵活的网站管理能力，支持信息栏目维护、采集字段自定义、通知书模板配置及生源信息管理，同时提供业务办理查询与流程配置工具。

如需要其他产品源码，可访问天天开源官网。
